Privacy Policy

ShieldVault Browser Extension

ShieldVault does not collect, store, transmit, or share any user data. Ever.

What ShieldVault Does

ShieldVault is a browser extension that detects API keys, tokens, and credentials in text you type or paste into AI chat interfaces. When a potential secret is detected, it is blocked before submission.

Data Collection

ShieldVault collects zero data. Specifically:

  • No personal information is collected
  • No browsing history is recorded
  • No text you type is stored or logged
  • No detected secrets are saved anywhere
  • No analytics or telemetry data is gathered
  • No cookies are used

Data Transmission

ShieldVault makes zero network requests. All detection happens locally in your browser using pattern matching. No data is ever sent to external servers.

Data Storage

ShieldVault uses session-only memory to track blocked events for display in the popup. This information:

  • Exists only in browser memory
  • Contains no actual secret values (only detection metadata like type and timestamp)
  • Is automatically deleted when you close your browser
  • Is never written to disk or persistent storage

Permissions

ShieldVault requests only the minimum permissions required:

  • Host permissions: Limited to specific AI chat platforms (ChatGPT, Claude, Gemini, Perplexity, Copilot)

Third Parties

ShieldVault does not share data with third parties because ShieldVault does not collect data. There are no advertising partners, analytics providers, or data brokers involved.

Changes to This Policy

If this privacy policy changes, the updated version will be posted here. The core commitment—no data collection—will not change.
